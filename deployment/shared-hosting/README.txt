PAKET DEPLOYMENT SHARED HOSTING

1. Ekstrak laravel_app.zip di public_html/laravel_app.
2. Ekstrak public_html.zip langsung di public_html.
3. Salin .env.production.example menjadi .env di laravel_app.
4. Isi APP_KEY dari .env lokal dan kredensial database Hostinger.
5. Pastikan PHP domain menggunakan versi 8.3 atau lebih baru.
6. Jangan menjalankan key:generate karena database memuat data terenkripsi.
